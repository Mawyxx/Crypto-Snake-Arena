# Slither.io Style Reference

Скачано с https://slither.io для референса стиля Crypto Snake Arena.

## Цвета и стили

### Body / фон страницы
- `background: #161c22` — тёмный синевато-серый

### Шрифты
- Основной: `'Lucida Sans Unicode', 'Lucida Grande', sans-serif`
- UI/кнопки: `'Arial', "Helvetica Neue", Helvetica, sans-serif`

### UI элементы (логин)
- Input text: `#e0e0ff` (светло-лавандовый)
- Placeholder nick: `#7070B0` (серо-фиолетовый)
- Placeholder victory: `#80C8FF` (голубой)
- Nick input bg: `#4C447c` (фиолетово-серый)
- Victory input bg: `#00A4CF` (циан)
- Tips: `#8068C0` (фиолетовый)
- Links: `#80FF80` (зелёный), `#8058D0` (фиолетовый)
- Footer: `#404880`

### Миникарта
- Fill: `#202630`
- Сегменты: `#404650`
- Тень: `#000000`, blur 12px

### Голова змеи
- Глаза: белые (#FFFFFF) с чёрным зрачком (#000000)
- Обводка: белая для своей, цветная для чужих

### Фон арены
- bg54.jpg — гексагональный паттерн
- URL: http://slither.io/s/bg54.jpg

### Рендеринг
- Canvas 2D или WebGL (Pixi.js)
- Еда: radial gradient с glow
- Змеи: line segments с цветом по body
